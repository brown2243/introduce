## 자기소개 Web 프로젝트

---

### 제작 의도

- 취업을 위한 자기어필(?)을 위해서 만들었습니다.
- 프로젝트 생성은 create-react-app을 사용했지만, 기능들을 바닐라JS로 만들었습니다.
- 이 프로젝트를 만들 당시 리액트로 만든 게시판 프로젝트는 있었지만, 바닐라 JS로 만들어 본 프로젝트가 없었기에, 바닐라 JS를 사용했습니다. (지금와서 보면 굳이...)

### 구성

##### Intro

SVG를 사용한 애니메이션

##### Content_skill

Skill 아이콘을 클릭하면, 할 수 있는 내용을 구체적으로 보여줍니다.

##### Content_Projects

작업했던 프로젝트의 요약입니다.
클릭하시면 슬라이드형태로 이미지를 보실수 있습니다.

##### Out_tro

기초 css 애니메이션
